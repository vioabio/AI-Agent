# 宝可梦对战与养成深度

#### 5V对项是什么意思，怎么孵蛋最快？
- **V** 指个体值31（满值），**5V对项** = 5项个体值31 + 无用的一项随机。
- **最快孵蛋流程**：1）用百变怪（6V最好）+目标宝可梦放在饲养屋；2）带上火焰之躯特性的宝可梦（如煤炭龟）加速孵化；3）每孵出一批替换更高V的父/母本；4）红线锁性格（不变之石）+ 红线遗传5项个体值。朱紫中可用"银色王冠"在冰柜镇的极限特训NPC处直接补个体值，省去孵蛋时间。

#### 努力值怎么分配？什么性格最好？
- **速攻型**（如烈咬陆鲨、苍炎刃鬼）：252攻击 + 252速度 + 4HP，性格爽朗（+速度 -特攻）或固执（+攻击 -特攻）。
- **特攻型**（如沙奈朵、水伊布）：252特攻 + 252速度 + 4HP，性格胆小（+速度 -攻击）或内敛（+特攻 -攻击）。
- **物盾型**（如钢铠鸦、吉利蛋）：252HP + 252防御 + 4特防，性格大胆（+防御 -攻击）。
- **特盾型**（如幸福蛋、洛托姆）：252HP + 252特防 + 4防御，性格温和（+特防 -攻击）。

朱紫中努力值可通过"力量道具"系列刷野生宝可梦快速获取，或直接喂"羽毛"和"营养饮料"。

#### 双打对战队伍怎么组？推荐几个热门体系
- **沙暴队**：班基拉斯（扬沙）+龙头地鼠（拨沙），加速+地震岩崩推队。
- **雨天队**：大嘴鸥（降雨）+海豚侠（全能形态）+刺龙王（悠游自如），水系爆破流。
- **空间队**：青铜钟/谜拟Q（开戏法空间）+超甲狂犀/嘎啦嘎啦，低速高攻碾压高速脆皮。
- **威吓轮转**：土地云灵兽+炽焰咆哮虎，双威吓削弱对方物攻手，控场压制。

#### 朱紫太晶团体战怎么打6星坑？
- 必须培养100级、努力值满、携带对应道具的宝可梦。
- **铁臂膀**（腹鼓+吸取拳）是万能打手，打妖精/冰/岩石/恶/钢/普通坑都稳。
- **玛力露丽**（腹鼓+嬉闹/水流裂破）适合打龙/格斗/火坑。
- 注意点：6星坑BOSS会清除自身负面状态（第3回合开始），不要拖太久。联网打建议带"大声咆哮"或"反射壁"辅助队友。

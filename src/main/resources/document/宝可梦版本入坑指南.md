# 宝可梦版本入坑指南

#### 新手应该从哪个宝可梦游戏开始？
- **Switch入门首选《剑/盾》**：操作友好、流程清晰、旷野地带自由探索。丹帝冠军战演出燃炸，DLC铠之孤岛+冠之雪原内容充实。
- **GBA经典《绿宝石》**：战斗边疆封神，双打机制完善，烈空坐剧情震撼。适合喜欢挑战性和怀旧的玩家。
- **不推荐从《朱/紫》入门**：虽然内容最丰富（开放世界+三条主线+太晶团体战），但掉帧和bug可能影响第一印象。

#### GBA红宝石、蓝宝石、绿宝石怎么选？
- **红宝石/蓝宝石**：基础版本，内容一致，仅限定的神兽不同（固拉多 vs 盖欧卡）。
- **绿宝石**：增强版，增加战斗边疆、双打战斗塔、烈空坐主线，是GBA世代最完整的版本。**推荐直接玩绿宝石**。

#### Switch上的朱和紫怎么选择？
- **朱**：限定宝可梦包括故勒顿（封面神）、振翼发、爬地翅等古代种悖谬宝可梦。
- **紫**：限定宝可梦包括密勒顿（封面神）、铁包袱、铁臂膀等未来种悖谬宝可梦。
- **推荐紫**：密勒顿的闪电冲刺赶路手感极佳，联网交换容易换到朱版限定。有条件建议双版本或订阅HOME传输。

#### 传说 Z-A 是什么，什么时候出？
传说系列新作，舞台在卡洛斯地区的密阿雷市（XY的米亚雷市），时间线为XY之后。
核心亮点：**Mega进化正式回归！** 预告片中出现了Mega喷火龙X和Mega沙奈朵。
预计2025年秋季发售，目前已知Mega蜥蜴王和Mega巨沼怪也会登场。
全图鉴宝可梦均可在密阿雷市的高楼和街区中捕捉，战斗系统结合了阿尔宙斯的即时动作元素。

#### 宝可梦HOME有什么用？
免费的云存储服务，可以在不同游戏之间传输宝可梦。
- 免费版：1个宝箱（30只），支持GTS交换和奇迹交换
- 高级版（年费）：200个宝箱（6000只），支持从3DS宝可梦银行迁移
- 核心用法：把剑盾的培育精灵传到朱紫PVP用，或收集全图鉴
